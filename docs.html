‚öôÔ∏è How It Works

Architecture Overview

Ada follows a client-server architecture with three main components:

Frontend
Vanilla JS, Tailwind CSS, CodeMirror
Backend
Python Flask, Gemini API
Database
Firebase Firestore

Request Flow

  1. User Input: You type a message or upload a file
  2. Token Generation: Firebase generates a secure ID token
  3. API Request: Frontend sends request to backend with token
  4. Token Verification: Backend verifies token using Firebase Admin SDK
  5. Gemini Processing: Backend sends your query to Gemini AI
  6. Streaming Response: AI response streams back in real-time
  7. Code Extraction: Frontend extracts code blocks and updates editor
  8. Firestore Save: Conversation saved to your personal chat history

üîê Authentication

Ada uses Firebase Authentication for secure, industry-standard user management.

Security Features

  • Server-Side Verification: All requests are validated using Firebase ID tokens
  • Automatic Token Refresh: Frontend automatically refreshes expired tokens
  • Protected API: Backend rejects unauthorized requests with 401 errors
  • Secure Storage: User data isolated in Firestore using UID-based paths

Login Methods

Method Description Benefits
Google OAuth One-click sign-in with Google account Fast, secure, no password needed

üîí Your Security Matters

All communication uses HTTPS. Passwords are hashed. API keys never exposed to frontend. Firebase handles all sensitive operations.

3. Code Block Extraction

Ada uses special markers to separate code from explanation:

// Ada's Response Format:
Here's the fixed code:

<<>>
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
<<>>

This algorithm has O(n¬≤) time complexity.

The frontend automatically:

  • Extracts code between markers
  • Updates the Code Editor
  • Renders explanation as markdown in chat

üîß Troubleshooting

‚ùå "Unauthorized" Error (401)

Cause: Token expired or invalid

Solution:

  • Sign out and sign in again
  • Clear browser cache and cookies
  • Token auto-refreshes on retry

‚è≥ Backend Slow to Respond

Cause: Render.com free tier spins down after inactivity

Solution:

  • First request may take 30-60 seconds (cold start)
  • Subsequent requests are fast
  • Set up UptimeRobot to keep backend alive

üìù Code Not Appearing in Editor

Cause: AI didn't include code markers or network issue

Solution:

  • Check if response includes code in chat
  • Manually copy code to editor
  • Refresh page and try again

üö´ "I am designed only for coding tasks"

Cause: Asked a non-coding question

Solution:

  • Ada only answers coding-related questions
  • Rephrase question to be code-specific
  • This is expected behavior

üì± Mobile Tab Not Switching

Cause: JavaScript error or browser issue

Solution:

  • Refresh the page
  • Clear browser cache
  • Try a different mobile browser
